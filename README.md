# Employee Management System (GUI)

## 📌 Project Overview

The Employee Management System is a GUI-based desktop application developed using Python and Tkinter.
It allows users to manage employee records efficiently including adding, removing, searching employees and calculating total payroll.

The application follows Object Oriented Programming (OOP) principles and uses file handling (pickle) to store employee data permanently.

---

## 🚀 Features

* Add new employees (Full Time / Part Time / Manager)
* Remove employee records
* Search employees by ID or Name
* Display all employee details
* Calculate Total Payroll
* GUI based interface using Tkinter
* Persistent data storage using Pickle
* Scrollable output display

---

## 🧠 Employee Types Supported

### ✅ Full Time Employee

* Fixed monthly salary

### ✅ Part Time Employee

* Salary calculated based on hours × rate

### ✅ Manager

* Salary = Base Salary + Bonus

## 📂 File Storage

Employee data is automatically saved in:

```
employees.pkl
```

This ensures data is not lost after closing the application.

---

## ▶️ How to Run the Project

1. Install Python (3.x)
2. Save the file as:

```
ems.py
```

3. Open terminal / command prompt

4. Run:

```
python ems.py
```
